# Raspberry Pi
Raspberry Pi 是由樹莓派基金會所開發的單板微型電腦(Single Board Computer)，其使用 Broadcom 出產的 ARM 架構晶片作為主要處理器，可以運作類 Unix 、 Android 、 FreeBSD 或是 Windows (ARM 版本的 Windows 11 甚至可在第四代主機板運行)等作業系統。
